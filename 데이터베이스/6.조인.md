# 조인
- **정의** : 두 개 이상의 테이블을 연결해 하나의 결과 집합으로 만드는 연산
- **20자** : 테이블간 관련 데이터를 연결하는 연산
- MySQL에선 JOIN, MongoDB에선 lookup 쿼리
  - lookup은 RDB에 비해 성능이 떨어지므로, 조인이 많으면 RDB를 써야한다

### 내부 조인 (Inner Join)
- 두 테이블에서 **조건에 일치하는 행만** 반환.
- 교집합 개념
```
SELECT * 
FROM A 
INNER JOIN B ON A.id = B.id;
```
