# 인덱스의 필요성
- 인덱스는 **데이터 검색 속도 향상**을 위해 사용된다.
- 책의 목차와 같다. 원하는 페이지를 빠르게 찾듯, 인덱스를 통해 DB는 전체 테이블을 스캔하지 않고 원하는 데이터를 효율적으로 찾을 수 있다.
- **없을 때**: Full Table Scan → `O(N)`
- **있을 때**: Index Lookup → `O(log N)`
