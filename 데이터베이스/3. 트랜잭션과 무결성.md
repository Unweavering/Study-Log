# 트랜잭션
- 데이터베이스에서 **논리적 작업 단위** (쿼리 묶음)
- ACID 원칙을 지켜야 함
## 원자성(Atomicity)
- 트랜잭션의 작업은 **모두 실행**되거나 **전혀 실행되지 않아야 한다 **(all or nothing)**
### 커밋 (Commit)
- 트랜잭션 정상 종료 후 데이터 영구 반영 (롤백 불가 → 되돌리려면 새 트랜잭션으로 반대 작업 수행)
