# 🔧 버블 정렬 (Bubble Sort)
## ✅ 개념
- 인접한 두 원소를 비교하여, 큰 값을 뒤로 보낸다.
- 거품이 수면 위로 떠오르 듯이, 가장 큰 수가 "거품처럼" 뒤로 떠오르는 방식.
## 📌 구현 로직
```
for (int i = 0; i < arr.length - 1; i++) {
    for (int j = 0; j < arr.length - i - 1; j++) {
        if (arr[j] > arr[j + 1]) {
            swap(arr, j, j + 1);
        }
    }
}
```
## ⏱️ 시간복잡도
- 최악, 평균: O(n²)
- 최선 (이미 정렬된 경우): O(n) (개선된 버블 정렬에서)
## 🧠 특징
- In-place, Stable
- 학습용으로 적절하나 실전에는 부적절
- 최적화 없이 사용 시 성능 최악
