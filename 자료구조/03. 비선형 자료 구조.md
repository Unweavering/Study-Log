# 그래프 (Graph)
- 정점(Vertex): 개체(노드)
- 간선(Edge): 정점 간 연결. 방향성 있을 수도 있음.
- 가중치(Weight): 간선에 부여된 비용/거리/시간.
- 👉 활용: SNS 친구 관계, 네트워크 라우팅, 최단 경로 알고리즘(Dijkstra, Floyd-Warshall).
