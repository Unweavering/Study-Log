# 🧠 1. 배열 (Array)
## ✅ 정의
- 메모리상에 **연속된 공간**에 데이터를 저장.
- 정해진 인덱스를 통해 O(1) 시간에 접근 가능.

- 리스트 형식 데이터 처리 (ex. 상품 목록)

# 🧱 3. 스택 (Stack)
## ✅ 정의
- **후입선출(LIFO)** 구조
- 마지막에 넣은 것이 먼저 나간다
## ✅ 주요 연산
- `push(item)`: 항목 추가
- `pop()`: 마지막 항목 제거
- `peek()`: 마지막 항목 조회
## ✅ 시간복잡도
![image](https://github.com/user-attachments/assets/43f4a077-9e24-40ad-9104-484a26eb538c)
## ✅ 사용 예시
- 재귀 호출 추적
- 괄호 유효성 검사
- DFS 구현
- Undo 기능
## ✅ 구현 방식
- 배열 기반 스택
- 연결리스트 기반 스택
